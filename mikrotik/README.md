# MikroTik Configs
Config files for [MikroTik](https://mikrotik.com/) network equipment.

### Management tools
- Download and install [WinBox](https://mikrotik.com/download) to manage devices, required for accessing MAC server

### Basic CLI commands
- SAFE mode: `CTRL + X`
- Print config: `/export`
